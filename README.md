**The Cosmic Collective Intelligence**

## Features
- Streaming responses powered by Grok (xAI)
- Custom HiveStar personality (hive-mind + stellar wisdom)
- Markdown support, copy buttons, example prompts
- Fully responsive futuristic UI
- Zero-config Vercel deployment

## Tech Stack
- Next.js 15 (App Router)
- TypeScript + Tailwind
- Vercel AI SDK + xAI Grok models

## Local Setup
1. `npm install`
2. Copy `.env.example` → `.env.local` and add your xAI key from https://console.x.ai
3. `npm run dev`

## Vercel Deployment (Recommended)
1. Push this repo to GitHub
2. Go to vercel.com → New Project → Import this repo
3. Add Environment Variable: `XAI_API_KEY` (same value as above)
4. Deploy → live in < 60 seconds
